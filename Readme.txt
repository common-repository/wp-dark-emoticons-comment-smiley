WP Dark Emoticons Comment Smiley is plugin will replace string like :D :( :) to emoticon and words filtering. For more, Edit this plugin manually.

== Description ==

This plugin will display a dark emoticons smiley icon in wordpress comment system.by replacing the familiar string such as :),: (,:p,:D etc. and word filtering.

No configuration at all. to add the word filtering, you must manually edit the emoticons.php tag / * WORD FILTER * /. 



== Installation ==

This section describes how to install the plugin and get it working.



1. Upload `wp-darkemoticons.zip` to the `/wp-content/plugins/` directory. You can do this using 'Upload' functionality provided in plugins section of your wordpress dashboard
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You are done.
